Bot de Telegram que permite consultar el clima de una ciudad utilizando reconocimiento de lenguaje natural, proporcionado por la API Wit.ai, para evitar el uso de comandos. Por ejemplo puede enviar el siguiente mensaje "¿Cómo está el clima en Buenos Aires?" o "¿Qué tiempo hace en Buenos Aires?" y el bot responderá con la información del clima en la ciudad, proporcionada por la API OpenWeatherMap.
